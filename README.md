# Bootcamp Banco Carrefour

Este projeto tem como objeto reunir os arquivos de estudos relativos ao bootcamp do banco carrefour em parceria com a DIO. Aqui vão estar presentes exercicios de aula, materiais adicionais e minhas anotações pessoais sobre a compreensão do conteudo do curso. Fique a vontade para colaborar e dar sugestões.
